## PART III: VILLAIN SELECTION CHECKLIST

Score each villain candidate using this weighted matrix:

### Scoring Matrix (Rate 1-10 for each criterion)

| Criterion | Weight | Villain A | Villain B | Villain C |
|-----------|--------|-----------|-----------|-----------|
| **Root Source** (Is it the underlying cause?) | 20% | | | |
| **Solvability** (Can your product defeat it?) | 20% | | | |
| **Specificity** (Clear enough to rally against?) | 15% | | | |
| **Relatability** (Does audience immediately recognize it?) | 15% | | | |
| **Authenticity** (Aligns with brand purpose?) | 15% | | | |
| **Market Size** (How many affected?) | 10% | | | |
| **Legal Safety** (Can you prove claims?) | 5% | | | |
| **TOTAL WEIGHTED SCORE** | 100% | | | |

**Calculation Example:**
- If Villain A scores 8/10 on Root Source (20% weight): 8 × 0.20 = 1.6
- Repeat for all criteria and sum for total score

**Selection Rule:** Villain with highest weighted score becomes your strategic enemy.

## PART IV: VILLAIN VALIDATION CHECKLIST

Before committing, ensure your chosen villain passes ALL criteria:

- [ ] **ROOT SOURCE:** It's the cause, not the symptom
- [ ] **SOLVABILITY:** Your product can credibly defeat or significantly mitigate it
- [ ] **SPECIFICITY:** Clear, concrete, visualizable (not abstract)
- [ ] **RELATABILITY:** Target audience immediately recognizes and scorns it
- [ ] **AUTHENTICITY:** Genuinely aligns with your brand purpose
- [ ] **SINGULARITY:** You have ONE primary villain (not multiple)
- [ ] **MEASURABILITY:** Can be quantified or proven (ideal but not mandatory)

**If ANY box is unchecked, reconsider your villain selection.**

## PART X: LEGAL & ETHICAL GUARDRAILS

### FTC Compliance Checklist

When making comparative claims:

- [ ] All claims are truthful and non-deceptive
- [ ] Claims based on objectively measurable features
- [ ] All data substantiated with credible sources
- [ ] No false or misleading disparagement
- [ ] Clear disclosure of any material connections
- [ ] Avoid naming competitors unless absolutely necessary
- [ ] If naming competitors, have legal review all claims

**Golden Rule:** Attack the STATUS QUO (systemic problem), not specific competitors by name.

---

### Ethical Boundaries

**AVOID:**
- [ ] Creating false fear or anxiety
- [ ] Exaggerating villain's danger beyond truth
- [ ] Blaming customers for their problems
- [ ] Making medical claims without substantiation
- [ ] Positioning product as "cure-all" solution
- [ ] Using villain to shame or stigmatize

**EMBRACE:**
- [ ] Authentic empathy for customer struggle
- [ ] Evidence-based villain identification
- [ ] Realistic promises about what product can achieve
- [ ] Externalization of blame (villain, not customer)
- [ ] Educational approach to villain explanation
- [ ] Transparent about product limitations

## FINAL IMPLEMENTATION NOTES

### Success Factors

**The THREE MUSTS for Villain Marketing:**

1. **SINGULAR FOCUS:** One villain, one fight, one clear enemy
2. **AUTHENTIC ALIGNMENT:** Villain must genuinely relate to your product's purpose
3. **CREDIBLE DEFEAT:** Your product must actually solve the villain problem

**Common Pitfalls:**

- Multiple villains (confusing narrative)
- Villain disconnected from product solution
- Overpromising on villain defeat
- Villain too abstract or vague
- Villain that blames the customer
- Abandoning villain too quickly

**Golden Rules:**

1. Attack the STATUS QUO, not competitors by name (scalable + legally safe)
2. Externalize blame (villain, not customer)
3. Simplify the fight (cognitive ease)
4. Build the tribe (Unity principle)
5. Prove you can win (credibility)
